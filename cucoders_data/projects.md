## 🧾 Sistema de Gestión y Punto de Venta

https://github.com/ElCapi1195/ElCapi1195/blob/main/caja%20cobro%20.png
https://github.com/ElCapi1195/ElCapi1195/blob/main/Dashboard%20Caja%20admin.png

**Descripción:** Aplicación de escritorio para la gestión de operaciones de caja, trabajadores, pagos, turnos, descuentos y reportes.

El sistema utiliza una arquitectura con una computadora administrativa principal y diferentes estaciones de caja conectadas. Está diseñado para centralizar la gestión del negocio, controlar las operaciones y mantener un historial detallado de la actividad.

Entre sus funcionalidades se incluyen:

- Gestión de usuarios y trabajadores.
- Operaciones de caja y cierres diarios.
- Control de pagos y salarios.
- Registro de descuentos a trabajadores.
- Historial y reportes.
- Comunicación entre administración y estaciones de caja.
- Sistema de licencias y control de dispositivos autorizado.

**Tecnologías:** Electron · TypeScript · Node.js · SQLite

**Estado:** 🚧 En desarrollo

## 🌐 BioBridge — Plataforma Corporativa Internacional

https://github.com/ElCapi1195/ElCapi1195/blob/main/home%20se%20ve%20mejor.png

**Descripción:** Plataforma web corporativa para Hebei BioBridge Medical Technology Co., Ltd., orientada a conectar innovación, empresas y oportunidades de negocio entre China, América Latina y mercados internacionales.

Participo en el proyecto como **Webmaster y responsable de Transferencia Tecnológica**, trabajando en la planificación y evolución tecnológica de la plataforma.

El proyecto está diseñado para crecer por fases:

- Plataforma corporativa internacional.
- Soporte para español, inglés y chino.
- Gestión de contenido mediante CMS.
- Futuras funcionalidades CRM.
- Integración de servicios digitales.
- Arquitectura preparada para expansión.

**Tecnologías:** Next.js · React · TypeScript · Tailwind CSS

**Rol:** Webmaster y Transferencia Tecnológica

**Estado:** 🚧 En desarrollo
