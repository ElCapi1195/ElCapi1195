**Descripción:** Aplicación de escritorio para la gestión de operaciones de caja, trabajadores, pagos, turnos, descuentos y reportes.

El sistema utiliza una arquitectura con una computadora administrativa principal y diferentes estaciones de caja conectadas. Está diseñado para centralizar la gestión del negocio, controlar las operaciones y mantener un historial detallado de la actividad.

Entre sus funcionalidades se incluyen:

- Gestión de usuarios y trabajadores.
- Operaciones de caja y cierres diarios.
- Control de pagos y salarios.
- Registro de descuentos a trabajadores.
- Historial y reportes.
- Comunicación entre administración y estaciones de caja.
- Sistema de licencias y control de dispositivos autorizado.

**Tecnologías:** Electron · TypeScript · Node.js · SQLite

**Estado:** 🚧 En desarrollo
